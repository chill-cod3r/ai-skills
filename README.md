# AI Skills

Reusable Claude Code skills for shipping software with audit-grade evidence.

## Skills

### /ship
End-to-end skill for planning, building, and shipping features incrementally. Covers architecture planning, PRD generation with vertical-slice user stories, implementation, proportional verification, and explicit acceptance gating.

See [ship.md](ship.md) for the full skill definition.

## Usage

Copy any skill file into your project's `.claude/skills/` directory to make it available as a slash command in Claude Code.
